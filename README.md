# 📧 Email Investigation Toolkit

A React-based cybersecurity tool designed to help analysts systematically investigate suspicious emails. It provides a structured checklist, severity tracking, and notes — all in one clean interface.

---

## 🚀 Features

- ✅ Step-by-step investigation checklist (Triage, Headers, Links, Attachments, and more)
- 🔴 Severity indicators — Critical, High, Medium, Low
- 📝 Notes section for logging findings
- 📊 Progress tracker across checklist sections
- 🎨 Dark-themed UI built for analyst workflows

---

## 🛠️ Tech Stack

- [React](https://reactjs.org/) (with Hooks)
- Custom CSS-in-JS styling (no external UI library needed)

---

## 📦 Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/YOUR_USERNAME/email-investigation-toolkit.git
cd email-investigation-toolkit
```

### 2. Install dependencies
```bash
npm install
```

### 3. Run the app
```bash
npm start
```

The app will open at `http://localhost:3000`

## 📸 Usage

1. Open the toolkit in your browser
2. Work through each checklist section as you investigate the email
3. Mark items as complete and log your findings in the notes panel
4. Use severity flags to prioritize your response
